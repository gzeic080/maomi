这里提供更多的优秀课程资源，以下项目任意挑选学习。
项目一:雷霆战机
还记得小时候玩过的雷霆战机吗？
在浩瀚的太空，你只身一人孤军奋战，前面涌来的除了陨石以外还有凶狠的敌机。
Scratch编程将完美复原这个游戏，让我们一起找回曾经的热血吧。
学该项目前你需要掌握以下知识：
Scratch角色导入、删除、控制
你将会学到以下知识：
  1. "克隆"的进阶使用
  2. "变量"的进阶使用
  3. 函数式编程思想
  4. 二进制与十进制的转换
  5. 算法进阶“斐波那契数列”

这是农夫与兔子的故事
相信你也能发挥自己的创造力，做出好玩的动画！
学该项目前你需要掌握以下知识：
零基础可学

  1. 制作丰富的简易动画
  2. 条件判断、循环、嵌套等语句的使用
  3. 特殊的事件：广播
  4. 成语故事的背景及含义
